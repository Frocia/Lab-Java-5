# Lab-Java-5
Лабораторная работа: Решение задач на Java
Описание проекта
Проект содержит решения 8 задач, реализованных в отдельных классах на Java. Каждое задание демонстрирует различные аспекты программирования: работа с файлами, обработка данных, использование Stream API и ООП.

Структура проекта
Классы задач:
CompetitionResult - хранение данных о результатах соревнований

Point - представление точки в двумерном пространстве

Line - представление линии между двумя точками

Polyline - представление ломаной линии

CompetitionProcessor - обработка результатов соревнований (Задание 4)

LetterCounter - подсчет уникальных букв в тексте (Задания 5.7 и 6.2)

PointProcessor - обработка точек и создание ломаной (Задания 7.1 и 7.2)

Main - главный класс с пользовательским интерфейсом

Задания и их решение
Задание 4: Результаты соревнований по многоборью
Описание: Обработка данных о результатах школьного многоборья с определением трех лучших участников.

Входные данные (competition.txt):

text
5
Иванова Мария 5 8 6 3
Петров Сергей 9 9 5 7
Сидорова Анна 10 8 7 9
Козлов Дмитрий 6 7 8 5
Смирнова Ольга 10 8 7 9
Ожидаемый вывод:

text
=== ЗАДАНИЕ 4: Результаты соревнований по многоборью ===
Лучшие участники:
Сидорова Анна 10 8 7 9 (Сумма: 34)
Смирнова Ольга 10 8 7 9 (Сумма: 34)
Петров Сергей 9 9 5 7 (Сумма: 30)
Задания 5.7 и 6.2: Подсчет разных букв в тексте
Описание: Определение количества уникальных русских букв в текстовом файле.

Входные данные (text.txt):

text
В этом тексте содержатся различные русские буквы. 
Некоторые из них повторяются несколько раз, 
а другие встречаются только один раз. 
Этот текст предназначен для подсчета количества разных букв.
Ожидаемый вывод:

text
=== ЗАДАНИЯ 5.7 и 6.2: Подсчет разных букв в тексте ===
Количество разных русских букв в тексте: 25
Задания 7.1 и 7.2: Обработка точек и создание ломаной
Описание: Обработка набора точек с использованием Stream API: удаление дубликатов, сортировка, преобразование координат и создание ломаной линии.

Исходные данные:

Точки: [{1.0;2.0}, {3.0;4.0}, {1.0;2.0}, {5.0;-6.0}, {2.0;8.0}, {5.0;-6.0}, {0.0;-3.0}]

Ожидаемый вывод:

text
=== ЗАДАНИЯ 7.1 и 7.2: Обработка точек и создание ломаной ===
Исходные точки: [{1.0;2.0}, {3.0;4.0}, {1.0;2.0}, {5.0;-6.0}, {2.0;8.0}, {5.0;-6.0}, {0.0;-3.0}]
Результирующая ломаная: Линия [{0.0;3.0},{1.0;2.0},{2.0;8.0},{3.0;4.0},{5.0;6.0}]
Демонстрация работы классов
Ожидаемый вывод при демонстрации:

text
=== ДЕМОНСТРАЦИЯ РАБОТЫ КЛАССОВ ===
Точка 1: {10.0;20.0}
Точка 2: {30.0;40.0}
Линия: Линия от {10.0;20.0} до {30.0;40.0}
Ломаная: Линия [{1.0;1.0},{2.0;3.0},{4.0;2.0}]
Результат соревнований: Иванова Мария 5 8 6 3 (Сумма: 22)
Требования к выполнению
✅ Все задачи оформлены в одном проекте в разных классах

✅ Реализована проверка входных данных

✅ Каждый класс содержит свойства, конструкторы и метод toString()

Особенности реализации
Stream API - использован для обработки коллекций и работы с потоками данных

Обработка исключений - реализована проверка корректности входных данных

Работа с файлами - чтение данных из текстовых файлов

Объектно-ориентированный подход - логическое разделение ответственности между классами

Интерактивное меню - удобный пользовательский интерфейс для тестирования функционала
